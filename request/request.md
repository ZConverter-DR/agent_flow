현재까지 완성한 프로젝트가 어떻게 동작하는지 모르겠어
일단 내 개발 배경은 java로 백엔드 프로젝트 팀장으로 진행해봄
기본적인 MCP 개념 및 API 구조에 대한 개념은 있음
다만 파이썬으로 프로젝트는 처음 해봐
내가 궁금한건 아래와 같아

1) 해당 프로젝트를 실제 실행해보려면 내 컴퓨터에 설정해야 하는 환경 셋팅
2) 실제 채팅에 프롬프트를 넣으면 코드 레벨에서 어떻게 flow가 흘러서 동작하는지 알고 싶음
3) 해당 프로젝트에서 RAG를 구축하려면 내가 선행적으로 갖추어야 하는 지식 알려줘 (EX: 파이썬 문법, 랭그래프, 랭체인 등)
